# Game Engine Readme
This folder contains all files relating to the game engine sub-project

## Map Generator
A very simple map generator is included inside `MapGenerator.html`. To start the map generator, just double click the file to open
in a browser. *Note:* A modern browser is required (Chrome/Firefox, something that supports ES6+).

How to use:
- Clicking in the grid (and dragging) will change the color of the buttons. Clicking multiple times will cycle the colours.
These represent either walls or spawn boxes
- Shift click/drag with erase the colour in the box
- To create the map, click the export button. This will create the map data in the format the same as the other maps found in resources/maps
